## 1. SYSTEM OVERVIEW
Sextant Protocol is a modular resilience intelligence framework designed to observe, simulate, and structure systemic stress across macro environments.

It functions as a structured interpretation layer for complex system behavior under uncertainty.

---

## 2. SYSTEM PURPOSE
The system exists to:
- Model scenario-based stress conditions
- Observe macro-level transitions
- Structure system state awareness
- Provide controlled simulation environments

It is not predictive. It is interpretive.

---

## 3. CORE ARCHITECTURE
The system is composed of four layers:

- INDEX LAYER → this repository (command and routing)
- EXECUTION LAYER → Operations Console
- SIMULATION LAYER → Scenario Simulator
